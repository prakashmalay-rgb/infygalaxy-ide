## ⚡ Get the .exe in 5 steps (no coding needed)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click + → New repository
2. Name it: infygalaxy-ide
3. Set it to Public
4. Click Create repository

### Step 3 — Upload these files
EASIEST WAY — use GitHub Desktop (https://desktop.github.com):
1. Install GitHub Desktop and sign in
2. File → Add Local Repository → select this folder
3. Click Publish Repository

OR via github.com website:
1. On your repo page click "uploading an existing file"
2. Drag ALL files and folders from this ZIP
3. Click Commit changes

### Step 4 — Watch it build (Actions tab)
1. Go to your repo → Actions tab
2. See "Build Windows Installer" running (~5 min)
3. Wait for the green checkmark

### Step 5 — Download your .exe
Go to Releases (right side of repo) → Download the .exe → Install it!

---

## Optional: GitHub OAuth setup
1. github.com/settings/developers → OAuth Apps → New OAuth App
2. Callback URL: http://localhost:7823/callback
3. Repo Settings → Secrets → Add OAUTH_CLIENT_ID and OAUTH_CLIENT_SECRET
4. Re-run build — GitHub sign-in will work in the app
